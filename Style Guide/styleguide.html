Default Button

Css Selector

button

button:focus, button:hover, button:active

About

These are the default buttons to be used throughout the website. They are dark blue with white text. When the mouse hovers over it, the button's background will change white with dark blue outline and text.

Example

Cancel Button

Css Selector

.cancel_button

.cancel_button:active, .cancel_button:hover, .cancel_button:focus

About

The cancel button would be used in the ordering process. When the user doesn't want to continue ordering anymore they are able to stop any time. The button will default with a light grey background and white text. It will hover red to alert the user they are canceling their order.

Example
